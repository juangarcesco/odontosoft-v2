# M1-02 — Seguridad y middlewares

Descripción:
Implementar `authMiddleware`, `roleMiddleware` y `rateLimiter` para proteger rutas.

Tareas:
- Middleware de verificación de JWT.
- Middleware de roles basado en `Usuario`.
- Límite de intentos en login con `rateLimiter`.

Criterios de aceptación:
- Rutas protegidas devuelven 401/403 según corresponda.

Estimación: 2d
